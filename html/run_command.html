Automatically run commands after Linux starts

Process

  1. Place your commands in a .sh file and put it into /etc/init.d. For example, the script in my case is /etc/init.d/mystartup.sh: 
    #!/bin/sh
sudo mkdir /media/C
sudo mount /dev/sda2 /media/C
  1. Run sudo chmod +x /etc/init.d/mystartup.sh to make it executable.
  2. Run sudo update-rc.d /etc/init.d/mystartup.sh. See man update-rc.d.
  3. Done. Good to go!
